
# 📘 Assignment: Games in Python

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and user input handling.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Implement a fully functional Hangman game that randomly selects a word from a predefined list and allows the player to guess letters until they either solve the word or exhaust their attempts.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in the format `_ _ _` (with revealed letters)
- Track and display the number of incorrect guesses remaining
- Detect when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages
